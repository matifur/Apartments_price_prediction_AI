# Sieć neuronowa do predykcji cen mieszkań

Krótki projekt pokazujący, jak zbudować prostą sieć neuronową do przewidywania cen mieszkań na podstawie cech nieruchomości. Główna część pracy znajduje się w notebooku [Apartmet_prices_neural_network.ipynb](/Users/macbook/MyProjects/Python/AI_2026_project/Apartmet_prices_neural_network.ipynb:1).

## Dane

- Plik danych: `data/Houses.csv`
- Źródło: Kaggle, House Prices in Poland
- Link do zbioru: https://www.kaggle.com/datasets/dawidcegielski/house-prices-in-poland/data

## Jak uruchomić projekt

1. Otwórz folder projektu w Visual Studio Code.
2. Upewnij się, że masz zainstalowanego Pythona 3.
3. Doinstaluj rozszerzenia VS Code:
   - `Python`
   - `Jupyter`
4. Zainstaluj biblioteki z pliku `requirements.txt`:

```bash
python3 -m pip install -r requirements.txt
```

5. Otwórz notebook [Apartmet_prices_neural_network.ipynb](/Users/macbook/MyProjects/Python/AI_2026_project/Apartmet_prices_neural_network.ipynb:1).
6. Wybierz aktywny interpreter lub kernel Pythona w prawym górnym rogu VS Code.
7. Uruchom komórki od góry do dołu, najlepiej przez `Run All`.

## Zawartość projektu

- `Apartmet_prices_neural_network.ipynb` - główny notebook z opisem, kodem, wynikami i wnioskami
- `data/Houses.csv` - plik z danymi wejściowymi
- `requirements.txt` - lista bibliotek potrzebnych do uruchomienia projektu
